## Overview
EcoLens is a community-driven environmental monitoring platform designed to empower citizens to report pollution incidents in real-time. This Android application allows users to capture photos of pollution hotspots, tag precise locations, and engage with the community to foster environmental awareness.

## Features ‚ú®
* **Pollution Reporting:** Capture or upload images, categorize pollution (Air, Water, Soil, Noise), and tag precise locations using Google Maps.
* **Real-Time Tracking:** Monitor the status of submitted reports (Pending/Resolved) and view resolution proofs uploaded by authorities.
* **Community Connect:** Real-time chat interface for users to discuss local environmental issues and collaborate.
* **Daily Tips:** Curated feed of sustainability advice and environmental tips to promote green habits.
* **User Profile:** Manage account details and view submission history.
